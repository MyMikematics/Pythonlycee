    \hypertarget{classe-de-seconde}{%
\subsection{Classe de seconde :}\label{classe-de-seconde}}

    Une consolidation des acquis du cycle 4 est proposée autour de deux
idées essentielles :

\begin{itemize}
\tightlist
\item
  la notion de fonction ;
\item
  la programmation comme production d'un texte dans un langage
  informatique.
\end{itemize}

Dans le cadre de cette activité, les élèves s'exercent à :

\begin{itemize}
\tightlist
\item
  décrire des algorithmes en langage naturel ou dans un langage de
  programmation ;
\item
  en réaliser quelques-uns à l'aide d'un programme simple écrit dans un
  langage de programmation textuel ;
\item
  interpréter, compléter ou modifier des algorithmes plus complexes.
\end{itemize}
